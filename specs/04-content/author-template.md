## Content Requirements

| Element | Specification |
|---------|---------------|
| **Name** | Full real name |
| **Bio** | 200-300 words |
| **Photo** | Professional headshot (real, not AI) |
| **Credentials** | Degrees, certifications |
| **Experience** | Years in field, specializations |
| **External Links** | LinkedIn, professional profiles |
| **Articles** | List all articles by author |

## EEAT Requirements

| Requirement | Specification |
|-------------|---------------|
| **Real Name** | Must use real name, not pseudonym |
| **Verifiable Credentials** | Links to LinkedIn, professional profiles |
| **Professional Photo** | High-quality headshot |
| **Expertise** | Minimum 3 years in relevant field |
| **Author Page** | Dedicated /author/{slug}/ page |

## SEO Requirements

| Element | Specification |
|---------|---------------|
| **Title Tag** | Author name + "Finance Expert" |
| **Meta Description** | 120-158 characters |
| **H1** | Author name |
| **Schema** | Person, BreadcrumbList |
| **Canonical** | Self-referencing |
| **Internal Links** | Link to all articles by author |

## Quality Checklist
