## Content Requirements

| Element | Specification |
|---------|---------------|
| **Title** | 50-60 characters, primary keyword first |
| **Meta Description** | 120-158 characters, includes CTA |
| **Introduction** | 100-150 words, hooks reader, includes keyword |
| **Body** | 2000-3000 words, clear sections, subheadings |
| **Internal Links** | 3-5 links to related content |
| **External Links** | 2-3 links to official sources |
| **Images** | 3-5 relevant images with alt text |
| **FAQ** | 3-5 questions with FAQ schema |
| **CTA** | Clear call-to-action at end |

## Writing Guidelines

### Tone

- Professional but approachable
- Jargon-free (explain complex terms)
- Authoritative but not condescending
- Helpful and educational

### Formatting

- Use short paragraphs (2-3 sentences)
- Use bullet points and numbered lists
- Use tables for comparisons
- Use bold for key points
- Use blockquotes for important information

### SEO

- Include primary keyword in first 100 words
- Use keyword variations throughout
- Include LSI keywords naturally
- Use descriptive headings (H2, H3)
- Include alt text for all images
