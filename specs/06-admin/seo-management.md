# SEO Management

> **Version:** 2.0.0  
> **Last Updated:** 2026-06-14

## Overview

The SEO management module provides tools to optimize search engine visibility across the site.

## Meta Editor

### Features

| Feature | Specification |
|---------|---------------|
| **Bulk Edit** | Edit meta for multiple pages at once |
| **Preview** | Live preview of Google search result |
| **Validation** | Character count and warnings |
| **Suggestions** | AI-powered meta suggestions |

### Fields

| Field | Specification |
|-------|---------------|
| **Meta Title** | 50-60 characters, keyword-optimized |
| **Meta Description** | 120-158 characters, includes CTA |
| **Canonical URL** | Self-referencing, HTTPS |
| **Robots Directive** | index/noindex, follow/nofollow |

## Open Graph Editor

### Features

| Feature | Specification |
|---------|---------------|
| **Preview** | Live preview of social share |
| **Fields** | og:title, og:description, og:image |
| **Image Upload** | Upload custom OG image |
| **Validation** | Dimensions and format check |

### Image Requirements

| Requirement | Specification |
|-------------|---------------|
| **Dimensions** | 1200x630px |
| **Format** | PNG |
| **Max Size** | 1MB |

## Sitemap Management

### Features

| Feature | Specification |
|---------|---------------|
| **View** | View current sitemap |
| **Regenerate** | Regenerate sitemap on demand |
| **Submit** | Submit to Google Search Console |
| **Validate** | Validate sitemap format |

## Redirect Manager

### Features

| Feature | Specification |
|---------|---------------|
| **Create** | Create new redirects |
| **Edit** | Edit existing redirects |
| **Delete** | Delete redirects |
| **Bulk** | Bulk create redirects |
| **Import** | Import redirects from CSV |

### Redirect Types

| Type | Usage |
|------|-------|
| **301** | Permanent redirect |
| **302** | Temporary redirect |

## Internal Link Manager

### Features

| Feature | Specification |
|---------|---------------|
| **Suggestions** | AI-powered link suggestions |
| **Broken Links** | Detect broken internal links |
| **Orphan Pages** | Detect pages with no internal links |
| **Bulk Update** | Bulk update internal links |
