# Analytics

> **Version:** 2.0.0  
> **Last Updated:** 2026-06-14

## Overview

The analytics module provides insights into site traffic, user behavior, and content performance.

## Dashboard

### Metrics

| Metric | Description |
|--------|-------------|
| **Sessions** | Total sessions in period |
| **Users** | Unique users in period |
| **Pageviews** | Total pageviews in period |
| **Bounce Rate** | Percentage of single-page sessions |
| **Avg. Session Duration** | Average time on site |
| **Pages per Session** | Average pages viewed per session |

### Date Range

| Option | Description |
|--------|-------------|
| **Last 7 days** | Quick selection |
| **Last 30 days** | Quick selection |
| **Last 90 days** | Quick selection |
| **Custom** | Custom date range |
