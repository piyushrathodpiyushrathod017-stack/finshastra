# Internal Linking Rules

> **Version:** 2.0.0  
> **Last Updated:** 2026-06-14

## Core Principles

1. **Relevance** — Links must be contextually relevant
2. **Natural** — Anchor text must read naturally
3. **Strategic** — Links must support SEO goals
4. **User-Focused** — Links must help users navigate

## Linking Rules

### Every Article Must

1. Link to its pillar page within the first 100 words
2. Link to 2-3 related articles within the same silo
3. Link to 1 article in a different silo (cross-linking, max 2)
4. Have at least 3-5 internal links total

### Pillar Pages Must

1. Link to ALL supporting articles in their cluster
2. Link to related programmatic pages
3. Link to comparison pages (if applicable)
4. Link to calculator tools (if applicable)

### Programmatic Pages Must

1. Link to their parent pillar page
2. Link to related bank/city pages
3. Link to relevant cluster articles
4. Link to comparison pages

### Author Pages Must

1. Link to all articles by that author
2. Link to author's other profiles (LinkedIn, etc.)

### Comparison Pages Must

1. Link to individual review pages for each product compared
2. Link to related comparison pages
3. Link to "best of" pages
4. Link to relevant calculators

## Anchor Text Guidelines

### Do

- Use natural keyword variations
- Use descriptive anchor text
- Use branded anchor text when appropriate
- Use question-based anchor text for FAQs

### Don't

- Use exact-match anchor text excessively
- Use "click here" or "read more"
- Use generic anchor text like "this article"
- Stuff keywords into anchor text

## Link Placement Rules

| Placement | Priority | Notes |
|-----------|----------|-------|
| First paragraph | High | Link to pillar page |
| Within content | High | Contextual links to related articles |
| FAQ section | Medium | Links to detailed articles |
| Related articles | Medium | Links to similar content |
| Footer | Low | Links to important pages |

## Forbidden Linking Patterns

| Pattern | Why Forbidden |
|---------|---------------|
| Link blocks with 20+ links | Link spam |
| "click here" anchor text | Poor UX, no SEO value |
| Orphan pages | Not discoverable by users or crawlers |
| Reciprocal link schemes | Manipulative |
| Hidden links | Deceptive |
| Links in comments | Spam |
| Links in footers with exact-match text | Over-optimization |

## Orphan Page Detection

Every page must be reachable within 3 clicks from homepage. Use tools to detect orphan pages:

1. Crawl site with Screaming Frog or similar
2. Check Google Search Console for indexed pages not in sitemap
3. Review internal link reports
4. Fix any orphan pages found
